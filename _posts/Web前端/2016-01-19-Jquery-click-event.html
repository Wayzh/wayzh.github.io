最近在写博客页面的时候，发现了Jquery的动态插入元素后，直接调用$(".class").click(function(){})绑定事件， 发现毫无反应。 比如： 


var data = ["web前端","Java后台"]
var length = data.length-1;
for(var i = 0 ;  i < length;i++){
	var cat = "<div><a class='btn btn-default btn-nav'>"+data[i]+"<a></div>";
	$(".panel-min-nav").append(cat);
}
$(".btn-nav").click(function(){
	console.log("test");//当点击上面动态添加的元素时，是无反应的
});

修正：

$(".panel-min-nav").on(
	"click",
	".btn-nav",
	function(){
		console.log("ok");	
});

自 jQuery 版本 1.7 起，on() 方法是 bind()、live() 和 delegate() 方法的新的替代品。推荐使用
